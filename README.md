# Static HTML Version (No npm required)

This folder is a pure HTML/CSS/JS build and runs on any static hosting.

## Files
- `index.html` and page files (`about.html`, `practice-areas.html`, `experience.html`, `consultation.html`, `contact.html`, `testimonials.html`, `blog.html`)
- `assets/css/style.css`
- `assets/js/main.js`
- `assets/images/advocate-anoop-pareek.svg`

## Free hosting options
- GitHub Pages
- Netlify (drag and drop this folder)
- Cloudflare Pages
- Vercel (static deploy)

## Notes
- Consultation form uses `mailto:` (opens visitor email app).
- For server-side form capture without backend coding, use Formspree/Getform and update form action.
- Update phone, email, office address, and image as needed.
